# HR-Analytics-Project
* Developed an interactive HR analytics dashboard using Microsoft Power BI to analyze employee attendance, work-from-home trends, and leave patterns.
* Utilized Power Query for data cleaning and transformation, and implemented DAX measures for KPI calculations including presence %, WFH %, and sick leave %.
* Designed insightful visualizations, interactive slicers, and trend analysis reports to support workforce monitoring and data-driven HR decision making.
